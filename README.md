# trakt_folder_sync

Плагин для Lampa. Двусторонняя синхронизация избранного с Trakt.tv (Trakt — источник истины):

- **Закладки** ↔ Trakt Watchlist
- **Брошено** ↔ личный список Trakt (выбирается в настройках)
- **Смотрю** ← Trakt upnext (с опциональным фильтром по дате последнего просмотра)

## Установка

Постоянная ссылка для Lampa:

```
https://mainsync-afk.github.io/test-lampa/trakt_folder_sync.js
```

Эта ссылка всегда ведёт на актуальную версию — обновлять её в Lampa не нужно.

## Настройки

В разделе «Trakt» появляются пункты:

- **Синхронизация папок с Trakt** — общий выключатель.
- **Период синхронизации папки «Смотрю»** — фильтр по дате последнего просмотра:
  - «Без ограничений» (по умолчанию) — все сериалы, которые Trakt считает upnext.
  - «1 день (отладка)» — только те, что смотрели последние сутки.
- **Список Trakt для папки «Брошено»** — выбор личного списка Trakt, с которым будет синхронизироваться папка «Брошено».
- **Синхронизировать папку «Брошено» из Trakt** — ручной запуск синхронизации брошенных (автоматически не срабатывает, в отличие от «Закладок» и «Смотрю»).

## Зависимости

Плагин `trakt_by_lampame` (или любой другой, предоставляющий `window.TraktTV.api`) должен быть установлен и активирован — через него идут все операции с Trakt API.

## История изменений

История версий ведётся через git-коммиты репозитория. Последние изменения:

- Фильтр папки «Смотрю» переработан: вместо выбора 7/14/30/60/90 дней — два режима «Без ограничений» и «1 день (отладка)». Причина: ограничение по max 90 дней отсекало сериалы, где последний просмотр был давно, а свежие серии вышли только что.
